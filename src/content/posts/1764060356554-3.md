Gemini API Key获取与配置教程

1.点击访问Google AI Studio，登陆你的谷歌账号！

[https://aistudio.google.com](https://aistudio.google.com/prompts/new_chat)

2.登陆后点击主页的“Get API Key”再点击“Create API Key”创建你的Gemini API Key

3.接下来你就会看到一个生成好的 API Key，点击 Copy 复制，然后保存好这个 Key，注意不要泄露自己的密钥

4.进入电子书转思维导图网站，点击主页的配置按钮，按照截图所示把复制好的Gemini API Key粘贴

[https://ebook2me.mind-elixir.com/](https://ebook2me.mind-elixir.com/)

5.上传你要解析的电子书，点击开始解析，过会儿你就可以获得自己需要的思维导图！
